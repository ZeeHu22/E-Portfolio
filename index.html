  • Bike Shop (v1)

    HTML, CSS, Javascript, JSON

    In the bike shop project, I developed a functional search engine that allows users to browse and filter available bicycles based on their preferences. The project leverages JavaScript promises to handle asynchronous data retrieval efficiently, ensuring that the search results are updated in real-time without affecting the user experience. By implementing promises, the project manages multiple data requests simultaneously, providing a smooth and responsive interface. This approach not only improved the overall performance of the website but also deepened my understanding of JavaScript's asynchronous capabilities, enhancing the user experience by ensuring faster, seamless interactions.
